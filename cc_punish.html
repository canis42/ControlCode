Sinsations CC Punish

Punish defines named punishments. Punishments may be initiated directly from the punish menu or by other modules that define certain conditions under which a punishment should be applied. Only one punishment may be applied at any time. The punishment "oc" is built in and can not be deleted. It also becomes the default if no other punishment is defined as default.

Punish Menu

Add
Add a punishment. In the first dialog you have to give a name for the punishment. In the second dialog you provide a textual definition of the punishment. Please see the chapter "Punishment Definition" below.
Change
Select a punishment name from the list of punishments. The retype the whole textual definition of the punishment with your changes. Please see the chapter "Punishment Definition" below.
Delete
Select a punishment name from the list of punishments. The punishment will be deleted.
Start
Select a punishment from the list of punishments. The Punishment will be played until you stop it.
Stop
Stop the currently applied punishment independent from how it has been applied.
Punish
Instantly apply the default punishment to the sub.
Default
Select the name of a punishment that should be the default punishment
Active
activate or deactivate all punishments

Punishment Definition

Punishment definitions are used as punishments if the punishment of the supplied name is activated.

Punishment definitions are given in Add- and Change menu items and in add and change commands.

A Punishment is defined by a line of text consisting of a comma separated list of definitions: definition1,definition2,definition3,...

A definition consists of a name and a value separated by a colon. The complete definition may look like this:
anim:~shock,sound:sndhum,rlv:dizzy,text:%s failed!,duration:300,sentence:I wil NOT do it again!

anim
Animation to be played as a punishment. Animations have to be in the Animate-prim. For example:~shock, ~shockold, ~nom, beautystand, belly, display, doggie, ...
sound
The sound is a sound file located in the main prim. Sounds supplied by the punish module are: sndau, sndhum, sndplasma, sndscream, sndvib.
text
The text is any text. it may contain %s which is replaced by the display name of the sub. The text is written to the chat.
particles
Not implemented yet. This is the name of a particle definition notecard.
duration
This is the duration of the punishment in seconds. The punishment is stopped after the given amount time (if not started using start). Default is 5 seconds if no sentence is given and 120 seconds if a sentence is given.
sentence
This is a text that has to be spoken in chat by the sub to end the punishment.

Punish Commands

Punish commands are issued on the chat line in the following form. Optionally a channel may be prependet like /1
<prefix> <command>

Names in pointed brackets have to be replaced by actual values.

menu pu
bring up the punish menu
pu add <name> <definitions>
Add a new punishment definition.
pu change <name> <definitions>
Change an existing punishment definition
pu delete <name>
Delete an existing punishment definition.
pu start <name>
Start a punishment until it is stopped manually
pu stop
Stop the currently applied punishment independent of how it is started.
pu punish
Start the default punishment using duration and sentence to stop it.
pu default <name>
Set the default to the given punishment name
pu on
Switch the punishment module on
pu off
Switch the punishment module off
